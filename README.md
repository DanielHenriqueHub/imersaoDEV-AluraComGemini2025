# Brasileirão Scout 2024 — Projeto da Imersão Alura + Google Gemini

Este projeto foi desenvolvido durante a Imersão Alura + Google Gemini, com o objetivo de praticar HTML, CSS, JavaScript e manipulação de dados em JSON, além de integrar IA no processo de criação.

O site permite buscar times do Brasileirão utilizando qualquer informação presente no arquivo JSON. A busca retorna cards dinâmicos contendo dados completos sobre cada time.

# Funcionalidades

## Busca inteligente: pesquisa por qualquer campo do JSON (nome, estado, estádio, sigla, situação etc.).

## Cards dinâmicos: ao clicar em "Buscar", os times correspondentes são exibidos com:

- Nome completo
- Escudo
- Ano de fundação
- Estado
- Estádio
- Títulos
- Lista de jogadores
- Link para a Wikipedia
- Página inicial estilizada com imagem ilustrativa e apresentação do projeto.
- Rodapé informativo com links úteis sobre a tabela atual do Brasileirão de 2025.


# Tecnologias Utilizadas

- HTML5 – Estrutura do site
- CSS3 – Estilização e layout responsivo
- JavaScript – Lógica da busca, manipulação do DOM e geração dos cards
- JSON – Base de dados dos times
- IA (ChatGPT/Gemini) – Auxílio na concepção e melhorias do projeto


# Como Funciona

- O usuário digita qualquer termo no campo de busca.
- O JavaScript percorre todos os campos do JSON.
- Os times compatíveis são exibidos em forma de card.
- Caso nada seja encontrado, uma mensagem aparece.



# Aviso Importante
As informações exibidas no site são apenas ilustrativas e podem estar desatualizadas ou imprecisas.

Este projeto tem finalidade exclusivamente educacional, criado para fins de estudo na imersão.
